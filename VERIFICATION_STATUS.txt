====================================================================================================
                                 VERIFICATION STRATEGIES
====================================================================================================

1. PYTHON SYNTHESIS (Computational Statements)
   ‚Ä¢ Triggers: "15 + 7 = 22", "sqrt(144) = 12", "25% of 80"
   ‚Ä¢ Method: LLM generates Python code ‚Üí Execute ‚Üí True/False
   ‚Ä¢ Speed: Fast (2-5 seconds)
   ‚Ä¢ Accuracy: Very High (95%+)

2. AUTOFORMALIZATION + LEAN (Logical/Mathematical Statements)
   ‚Ä¢ Triggers: "For all n...", "Theorem: ...", "Let f be continuous..."
   ‚Ä¢ Method: Autoformalize to Lean ‚Üí Theorem prover verification
   ‚Ä¢ Speed: Slower (10-30 seconds)
   ‚Ä¢ Accuracy: Very High (95%+) when formalization succeeds

3. LLM JUDGE (General Statements)
   ‚Ä¢ Triggers: "Therefore...", "This follows from...", "We conclude..."
   ‚Ä¢ Method: LLM evaluates logical soundness with context
   ‚Ä¢ Speed: Moderate (2-4 seconds)
   ‚Ä¢ Accuracy: Good (80-90%)

from openai import OpenAI
from veriform import AgenticVerifier

# Initialize
client = OpenAI(api_key="your-key")
verifier = AgenticVerifier(client, "gpt-4", "openai", verbose=True)

# Verify (automatic routing)
result = verifier.verify("15 + 7 = 22")
print(f"Correct: {result.is_correct}")      # True
print(f"Strategy: {result.strategy_used}")  # PYTHON_SYNTHESIS
print(f"Confidence: {result.confidence}")   # 0.9

# With context
result = verifier.verify(
    statement="Therefore, the total is $50",
    context=["Item A costs $30", "Item B costs $20"],
    problem_statement="Calculate total"
)
